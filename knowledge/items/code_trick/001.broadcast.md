## 所有能广播的情况

1. 形状完全一样 → 肯定能广播

比如：(2,3) 和 (2,3), (5,4,3) 和 (5,4,3)

**一模一样，直接算。**


2. 某一维是 1，另一维不是 1 → 能广播
比如：(1,3) 和 (2,3), (2,1) 和 (2,3), (1,1,8) 和 (2,3,8)

**只要是 1，就能扩成对方的大小。**

3. 维度数量不一样，但后面能对上 → 能广播
少的维度**自动往前补 1**，再继续比。

比如：(8,) 和 (2,3,8), (3,) 和 (5,4,3), (1,) 和 (10,20)

**后面对上就行，前面缺多少都补 1。**


4. 标量（0维）和任何张量 → 都能广播
标量相当于 shape=()，会自动补成：(1,1,1,...)，再扩成一样大。

比如：5 和 (2,3)，3.14 和 (5,4,3)
